ABOUT

Heroes Battle is an auto chess midcore game focused on strategy and merging your units. A roulette system decides which units will be available to be added to your squad every round making every run different.

MY WORK ON IT

This game is made in unity 3D. I developed the prototype of the project and once it officially became a viable project and a team was established I joined and developed most of the core gameplay and meta systems.

Main work on the project:

  • Character AI
  • Ability system
  • Roulette Sytem
  • Navigation system
  • Level up system
  • Challenge system
  • In game UI
  • Game mode variations core logic
